Adulthood

This set was the start of me falling back in love with LEGO as an adult. It is the most fun build ever and it's full of so many cool features including a sushi restaurant, comicbook store, a soda vending machine. It was a very big set and came out in conjunction with the Lego Ninjago Movie.

PRICE: It was released in 2017 at a price of €299. Today, the set sells for over €600 if new and unboxed. However, the great thing is even a used set will sell for over €500. You get to make it, play with it and then sell it on for a tidy profit.

You may have noticed this set has a striking similarity to the first set on the Childhood page. Well, yes. This spaceship took inspiration from that original set and went XL and a bit crazy. You can find a very funny minifigure called Benny with a cracked helmet. This was a nod to the flawed design of the original space helmet.

PRICE: On release in 2014, it cost €100 to buy. Nowadays, you'll pick up a new set for €175. Obviously, I was the only one who thought this set was cool.

This Creator Expert set was aimed directly at adult fans of lego(AFOLs). It was quite a technical build but the designers really crammed loads of interesting parts and techniques into the build that made it so appealing to me and to many others.

PRICE: Released in 2011 for €98, you can still pick a new one up for relatveily cheap(€133).
